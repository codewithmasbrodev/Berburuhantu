# 👻 Berburu Hantu — Nusantara (v2.0 Full-Stack)

Game mobile berburu hantu legendaris Indonesia.

## Setup Backend (mode online)
1. Buat project di https://supabase.com
2. Authentication → Sign In / Providers → Anonymous Sign-Ins → ENABLE
3. SQL Editor → paste `supabase/schema.sql` → RUN
4. Settings → API → salin URL & anon key → tempel ke `config.js`

## Deploy ke Vercel
Seret folder ini ke https://vercel.com/new → Deploy.
